Exporter tes données

Méthode recommandée : copier-coller

La plus fiable sur iPhone. Copie le texte, puis colle-le directement dans Réglages → Importer depuis le mobile sur l'ordinateur (champ « coller le JSON »).

Autres méthodes

Moins fiables selon les versions d'iOS — à essayer si le copier-coller ne te convient pas.

Voir le JSON brut

Marque les cartes et séances comme exportées une fois que tu es sûr d'avoir bien transféré tes données (copié-collé, partagé ou téléchargé) — ça ne les supprime pas, elles restent visibles dans « Déjà exportées » et tu peux toujours les effacer manuellement.
